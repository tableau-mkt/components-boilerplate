Boxes & Containers

Used as wrappers to apply certain styles around elements.

Padded box

Adds padding to a container element. The .box--dark-gray-1 class is used to differentiate the box from the background. You will usually want to use a padded box on a container with a background color.

Marginless Stretchy Boxes

Percentage width boxes that butt up against eachother with no margin. Typically, a collored background would be applied with a .box--{color} class. In a mobile layout, all stretchy boxes default to stacking one ontop of the other and in a tablet layout stretchy boxes default to a 50% split. A .box__stretchy--tablet-full class can be applied to a box to force it to be full-width on a tablet. This is useful when there are an odd number of boxes and you want to fill the extra blank space.
