Navigation

Navigational tools used across the site

Main Menu

The main navigational menu for the site.

Example

  • About
  • Content
  • Expo Hall
  • FAQ

Markup:

<div class="l-menu box--dark-blue">
  <ul class="menu">
    <li class="first leaf active-trail"><a href="#">About</a></li>
    <li class="leaf"><a href="#">Content</a></li>
    <li class="leaf"><a href="#">Expo Hall</a></li>
    <li class="leaf"><a href="#">FAQ</a></li>
  </ul>
</div>

Sidebar Nav

Jump-nav used in the sidebars to navigate between sub-sections of a single page

Example

  • Section 1
  • Section 2
  • Section 3

Markup:

<div class="sidebar--nav grid-3">
  <span class="sidebar--nav--toggle"><i class="icon-triangle-down"></i></span>
  <ul>
    <li class-"active"><a href="#section-1">Section 1</a></li>
    <li><a href="#section-2">Section 2</a></li>
    <li><a href="#section-3">Section 3</a></li>
  </ul>
</div>
