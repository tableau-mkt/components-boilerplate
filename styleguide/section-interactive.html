Content Flyout

Simple tool to make content hidden and flyout when a trigger is clicked.

Trigger Options:

  • data-flyout-target="[CONTENT ID]" - (required) - The ID of the container around the flyout content
  • data-flyout-scroll-down="true" - (optional) - Whether or not clicking this trigger should scroll down to the top of the flyout content container. Useful when the trigger referencing a flyout section lower down on the page

Content Reveal

Simple tool to make content hidden and revealed when a trigger is clicked. The trigger element can be abywhere on the page and can have the following options set as data attributes:

Trigger Options:

  • data-reveal-target="[CONTENT ID]" - (required) - The ID of the container around the content to be revealed
  • data-reveal-hide-text="[REVEAL HIDE TEXT]" - (optional) - Text to swap into the trigger link while the reveal is in the open state.
  • data-reveal-close-link="false" - (optional) - Set this to disable the default close link within the reveal content container
  • data-reveal-scroll="[trigger|target]" - (optional) - Smooth-scroll to either the top of the trigger or the target element when the content is revealed.
  • data-reveal-media="video" - (optional) - Enable play/pause controle when opening/closing reveal content that contains a brightcove video.

The .reveal__content container must have an id attribute that matches the data-reveal-target.

Example

Reveal Content

Markup

<div id="curtain">
  <p>
      <a href="#" class="reveal__trigger cta" data-reveal-target="content-container" data-reveal-media="markup" data-reveal-type="false" data-reveal-hide-text="Hide Content" data-reveal-curtain="curtain">Reveal Content</a>
  </p>
</div>
