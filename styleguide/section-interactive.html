Interactive Components

This section displays examples of components that have interactivity. Most Interactivity is defined via Javascript.

Content Reveal

Simple tool to make content hidden and revealed when a trigger is clicked. It doesn't matter where on the page the trigger and target content are, but the trigger element must target the content via a data-reveal-target="target-id" attribute. The .reveal__content container must have am id attribute that matches the data-reveal-target.

Example

Reveal Content

Markup:

<div id="curtain">
  <p>
      <a href="#" class="reveal__trigger cta" data-reveal-target="content-container" data-reveal-media="markup" data-reveal-type="false" data-reveal-curtain="curtain">Reveal Content</a>
  </p>
</div>

<div class="reveal__content" id="content-container">
  <img src='http://placecage.com/600/400'>
</div>
