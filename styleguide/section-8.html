Content Lists

Lists of content that generally have a title, thumbnail and/or a description and link through to the full version of the content.

Basic Teaser List

A basic list of teasers, used for things like a list of blog posts.

Examples

Default styling

Listed Item Title

Semper. Recusandae, vestibulum! Viverra cupidatat nostrud similique class mollis. Eius eget, nam! Pharetra?

Variation with a colored box around the title

.teaser-list--colored-title

Listed Item Title

Semper. Recusandae, vestibulum! Viverra cupidatat nostrud similique class mollis. Eius eget, nam! Pharetra?

Markup:

<div class="teaser-list grid-4 {$modifiers}">
  <a href="#">
    <div class="teaser-list__thumbnail">
      <img src="http://placecage.com/300/200">
    </div>
    <div class="teaser-list__info">
      <h2 class="teaser-list__title">
        Listed Item Title
      </h2>
      <p class="teaser-list__description">
        Semper. Recusandae, vestibulum! Viverra cupidatat nostrud similique class mollis. Eius eget, nam! Pharetra?
      </p>
    </div>
  </a>
</div>

Full-width Teaser List

A full-width list of teasers, used for things like a list of blog posts.

Example

Blog Post or something

Friday, April 10, 2015 - Make Me Smarter

Semper. Recusandae, vestibulum! Viverra cupidatat nostrud similique class mollis. Eius eget, nam! Pharetra? Litora condimentum expedita amet numquam, adipiscing pulvinar at interdum fringilla purus possimus fames.

Markup:

<div class="teaser-list teaser-list--full-width">
  <div class="teaser-list__thumbnail grid-3">
    <a href="#">
      <img src="http://placecage.com/223/223">
    </a>
  </div>
  <div class="teaser-list__info grid-9 grid-last">
    <h2 class="teaser-list__title">
      <a href="#">Blog Post or something</a>
    </h2>
    <div class="teaser-list__secondary-label">
      <span>Friday, April 10, 2015</span> - <span>Make Me Smarter</span>
    </div>
    <p class="teaser-list__description">
      Semper. Recusandae, vestibulum! Viverra cupidatat nostrud similique class mollis. Eius eget, nam! Pharetra? Litora condimentum expedita amet numquam, adipiscing pulvinar at interdum fringilla purus possimus fames.
    </p>
    <div class="class="teaser-list__read-more">
      <a href="/blog/another-post">Read More</a>
    </div>
  </div>
</div>
