Grids

Used for creating a columned layout. To use these classes, just pick the number of columns wide you want your element to be relative to the total number of columns. The default is 12 columns. For example, you can use .grid-4 to make an element 4 columns wide followed by an element with .grid-8 to take up the remaining eight columns of the container's width.
