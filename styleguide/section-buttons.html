Buttons

Buttons and CTAs used to draw more attention than a standard text link.

Secondary CTA

Secondary CTA buttons (Stroked Orange) should be used when multiple page actions are present, or somewhat important page actions are required.

Example

Download Trial More Customer Stories

Markup:

  <a class="cta cta--secondary  [modifier class]" href="#">
    Download Trial
  </a>
  <a class="cta cta--secondary  [modifier class]" href="#">
    More Customer Stories
  </a>

Tertiary CTA

Tertiary CTA buttons (Stroked alternate) should be used when a utility action, or an action not directly related to the page is required. When hovering over a Stroked button the copy is underlined.

Example

Free Trial

Markup:

  <a class="cta cta--tertiary  [modifier class]" href="#">
    Free Trial
  </a>
